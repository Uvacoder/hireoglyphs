# Hireoglyphs

## Naming

Symbols

`svg--{name}-{style}`

eg. `svg-image-outline`

Shapes

`svg--{shape}-{symbol}-{style}`

eg. `svg-circle-warning-outline`

## Design

### Settings

| Property          | Value           |
| ----------------- | --------------- |
| `line-cap`        | `rounded`       |
| `line-join`       | `rounded`       |
| `stroke-width`    | `2px`           |
| `border-radius`   | `2px`           |
